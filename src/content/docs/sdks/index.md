# Official SDKs

We are proud to offer official SDKs for interacting with ODict dictionaries from various programming languages. 

Currently, the following platforms are supported:

- [Go](./sdks/go)
- [Java, Kotlin, Scala, etc.](./sdks/jvm)
- [Node.js](./sdks/node)
- [Python](./sdks/python)
