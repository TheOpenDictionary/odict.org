We put together a nice table to illustrate.

|                                       | ODict | TEI  | XDXF | DICT | OntoLex |

| First released                        | 2017  | 1987 | 2006 | 1997 | 2016    |

| Features built-in indexing and lookup | ✅     | ❌    | ❌    | ✅    | ✅       |
